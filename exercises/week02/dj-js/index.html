DJ JS

Instructions:

  • Hover mouse over box to turn it blue.
  • Hold down mouse button on box to turn it red.
  • Release mouse button to turn it yellow.
  • Double click box to turn it green.
  • Scroll anywhere on page to turn box orange.
  • Press the key of the first letter of the color you want the box to change to
    • 'b' = blue
    • 'r' = red
    • 'y' = yellow
    • 'g' = green
    • 'o' = orange
